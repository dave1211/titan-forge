# Titan Forge

Build from anywhere. Launch from anything.

Titan Forge is a device-first website builder for phones, tablets, laptops, desktops, and offline-first workflows.

## MVP

- Landing page
- Guardian Space dashboard
- Mobile-first site builder
- Block editor
- Preview flow
- Tanner Terminal Lite
- Version 2 foundation
- Immigration AU module scaffold

## Stack

- Vite
- React
- Tailwind CSS
- Lucide icons
- Framer Motion

## Scripts

```bash
npm install
npm run dev
npm run build
```
